To download all twitter resources:
 1. Install the extension "Save All Resources" from here:
https://chrome.google.com/webstore/detail/save-all-resources/abpdnfjocnmdomablahdcfnoggeeiedb
 2. Open twitter.com and open inspector (Ctrl + Shift + I)
 3. Go to "ResourcesSaver" tab
 4. Check all the checkboxes
 5. Click "Save all resources"
 6. Move to this folder - git will highlight all the updates
